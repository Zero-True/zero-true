<Card title="Example Usage" icon="code">
  ```python
import zero_true as zt

# Create a DataFrame component
sample_dataframe = zt.DataFrame(
    id = 'sample_dataframe',  # Unique id for a component
    headers = [],  # List of column headers for the DataFrame
    items = [],  # List of items to be displayed in the DataFrame
    multi_sort = True,  # Enable or disable multi-sort on the DataFrame
    search = '',  # Create a text_input component search = zt.text_input(id='search') before to filter the DataFrame items
)

```
