<Accordion title="hint">
        **hint (string):** Hint text for the number input
</Accordion>

        **value (integer | float):** The input number value
</Accordion>
<Accordion title="label">
        **label (string):** Label for the number input
</Accordion>
<Accordion title="readonly">
        **readonly (boolean):** If true, the input is read-only
</Accordion>
<Accordion title="disabled">
        **disabled (boolean):** If true, the input is disabled
</Accordion>
<Accordion title="type">
        **type (string):** Ensures that only numbers are accepted on the frontend
</Accordion>
<Accordion title="step">
        **step (integer | float):** The number to increment or decrement by
</Accordion>

   </AccordionGroup>

<Card title="Example Usage" icon="code">
  ```python
import zero_true as zt

sample_numberinput = zt.NumberInput(
    id = 'sample_numberinput',  # Unique id for a component
    hint = 'Press Enter to Submit',  # Hint text for the number input
    value = 0,  # The input number value
    label = '',  # Label for the number input
    readonly = False,  # If true, the input is read-only
    disabled = False,  # If true, the input is disabled
    type = 'number',  # Ensures that only numbers are accepted on the frontend
    step = 1,  # The number to increment or decrement by
)

```
